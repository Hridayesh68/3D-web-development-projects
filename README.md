# 3D Web Development Projects using Blender

A collection of **3D web development projects** created using **Blender** and integrated into web applications.
This repository contains models, scenes, and examples demonstrating how **3D assets can be used in modern web development**.

---

## 📌 Overview

This repository focuses on:

* Creating **3D models and scenes in Blender**
* Exporting models for **web applications**
* Using **WebGL / Three.js / JavaScript frameworks** to render 3D content
* Building **interactive 3D experiences in the browser**

The projects demonstrate how Blender can be used alongside web technologies to create immersive and interactive websites.

---

## 🛠 Technologies Used

* **Blender** – 3D modeling and scene creation
* **Three.js** – Rendering 3D graphics in the browser
* **JavaScript** – Interaction and animations
* **HTML5 & CSS3** – Web page structure and styling
* **WebGL** – Hardware-accelerated graphics for the web

---

## 📂 Project Structure

```
3d-web-development-projects/
│
├── models/          # Blender models and exported files (GLB, FBX, OBJ)
├── scenes/          # Blender scene files (.blend)
├── web/             # Web implementation of 3D scenes
├── assets/          # Textures, materials, and resources
└── README.md        # Project documentation
```

## 🚀 Getting Started

### 1️⃣ Clone the repository

```
git clone https://github.com/hridayesh68/3d-web-development-projects.git
```

### 2️⃣ Open the project

Navigate into the folder:

```
cd 3d-web-development-projects
```

### 3️⃣ Run the web project

Open the HTML file in your browser or run a local server.

Example:

```
npx serve
```
